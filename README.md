# Template Restaurant Website

A modern, elegant restaurant website featuring a sophisticated design with full menu showcase, customer reviews, and reservation system. Built with HTML5, CSS3, and Tailwind CSS for a responsive and professional dining experience.

![Hero Section](https://hebbkx1anhila5yf.public.blob.vercel-storage.com/image-ScQBv0aXBzseEFVw8Amm3Ccsff9fE9.png)

## Features

### Interactive Menu System
- **Full Menu Display** - Comprehensive menu with high-quality food photography
- **Detailed Descriptions** - Each dish includes ingredients and preparation details
- **Price Information** - Clear pricing with serving size indicators
- **Category Badges** - Chef's Choice, Signature, Popular, and Best Seller tags
- **Fullscreen Image Modal** - Click any food image to view in fullscreen

![Menu Showcase](https://hebbkx1anhila5yf.public.blob.vercel-storage.com/image-riAaAHFny82y3Pn4waIxXLzikHGX4F.png)

### Customer Reviews Section
- **Verified Testimonials** - Real customer feedback with verification badges
- **Star Ratings** - 5-star rating system with average score display
- **Review Statistics** - Shows 4.9/5.0 rating from 2,500+ customers
- **Professional Illustrations** - Custom SVG graphics for visual appeal

![Customer Reviews](https://hebbkx1anhila5yf.public.blob.vercel-storage.com/image-RIV1Ayab2tFyZvPAxKi5dimp8kLqJy.png)

### Design Highlights
- **Responsive Layout** - Mobile-first design that works on all devices
- **Smooth Animations** - Hover effects and transitions throughout
- **Professional Typography** - Elegant font pairings for readability
- **Color Palette** - Warm, inviting colors perfect for fine dining
- **SVG Illustrations** - Custom graphics for wine tasting, chef, and reviews

## Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Custom styling and animations
- **Tailwind CSS** - Utility-first CSS framework via CDN
- **JavaScript** - Interactive modal and smooth scrolling
- **SVG Graphics** - Scalable vector illustrations

## Installation

1. Clone or download the repository
2. Open `index.html` in your web browser
3. No build process or dependencies required

\`\`\`bash

## Customization

### Colors
The color scheme uses warm, elegant tones:
- Primary: `#7C2D12` (Deep Brown)
- Secondary: `#D97706` (Golden Amber)
- Accent: `#92400E` (Rich Brown)
- Background: `#FEF3C7` (Cream)

### Fonts
- Headings: Playfair Display (serif)
- Body: Inter (sans-serif)

### Images
Replace images in `assets/images/` with your own:
- Recommended size: 1200x800px for food photos
- Format: JPG for photos, SVG for illustrations
- Optimize images for web performance
